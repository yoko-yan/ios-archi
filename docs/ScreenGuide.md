# 画面構成ガイド

このドキュメントは、UIテストを作成する際のコンテキストとして使用されます。

## アプリ概要

Minecraftワールド管理アプリ。写真からシード値や座標を認識する機能を持つ。

## 実装クラス対比表

画面の役割と実装クラス名の対応関係。クラス名変更時はこのセクションのみ更新。

| 画面の役割 | 実装クラス名 | セル/コンポーネント |
|-----------|-------------|-------------------|
| タイムライン | TimeLineView | TimeLineCell |
| 写真グリッド | SpotListView | - |
| ワールドリスト | WorldListView | WorldListCell |
| アイテム詳細画面 | ItemDetailView | - |
| アイテム編集画面 | ItemEditView | - |
| ワールド詳細画面 | WorldDetailView | - |
| ワールド編集画面 | WorldEditView | - |
| アプリ情報画面 | AppInfoView | - |

## タブ構成

| タブ | タイトル | メイン表示 | フローティングボタン |
|------|----------|------------|---------------------|
| ホーム | "ホーム" | タイムライン（リスト形式） | 写真アイコン+プラス → アイテム編集画面（シート） |
| 写真 | "写真一覧" | 写真グリッド（グリッド形式） | 写真アイコン+プラス → アイテム編集画面（シート） |
| ワールド | "ワールド一覧" | ワールドリスト（リスト形式） | 地球アイコン+プラス → ワールド編集画面（シート） |

## タブ別機能

### ホーム

- **ナビゲーション右上**: 情報ボタン（iアイコン）→ アプリ情報画面（モーダル）
- **リストアイテムタップ**: アイテム詳細画面へ遷移
- **Pull to Refresh**: リスト更新

### 写真

- **フィルター**: チェックボックス「画像を持っているアイテムのみ表示」
- **グリッドアイテムタップ**: アイテム詳細画面へ遷移
- **Pull to Refresh**: グリッド更新

### ワールド

- **リストアイテムタップ**: ワールド詳細画面へ遷移
- **左スワイプ**: 削除ボタン表示 → 削除確認アラート
- **Pull to Refresh**: リスト更新

## 共通画面

| 画面の役割 | タイトル | 主要機能 |
|-----------|---------|---------|
| アイテム詳細画面 | - | 写真/シード値/座標の表示、編集・削除操作 |
| アイテム編集画面 | 新規: "スポットを登録"<br>編集: "スポットを変更する" | 写真選択、シード値/座標入力、写真から自動認識 |
| ワールド詳細画面 | - | ワールド情報表示、バイオーム検索 |
| ワールド編集画面 | 新規: "ワールドを登録する"<br>編集: "ワールドを変更する" | ワールド名入力、シート表示（.medium/.large） |
| アプリ情報画面 | - | モーダル表示、×ボタンで閉じる、バージョン/ビルドバージョン表示 |

## UI要素

### フローティングボタン
各タブの右下に配置。タップで新規登録画面をシート表示。

### リスト/グリッド表示
- **ホーム/ワールド**: リスト形式
- **写真**: グリッド形式（最小80px）

### Pull to Refresh
全タブでサポート。下スワイプでデータ再読み込み。
